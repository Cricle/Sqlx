Example: Same Code, Different Databases

// Define once [Query("SELECT * FROM users WHERE name {{startswith}} {{limit 10}}")] Task<List<User>> FindByName(string prefix); // Auto-generated SQL adapts: SQL Server: WHERE name LIKE CONCAT(@prefix, '%') TOP 10 MySQL: WHERE name LIKE CONCAT(@prefix, '%') LIMIT 10 PostgreSQL: WHERE name LIKE @prefix || '%' LIMIT 10 SQLite: WHERE name LIKE @prefix || '%' LIMIT 10 Oracle: WHERE name LIKE @prefix || '%' AND ROWNUM <= 10

üìñ API Documentation

Quick reference for common APIs

[RepositoryFor(typeof(Entity))]

Marks interface as repository. Triggers source generation.

[RepositoryFor(typeof(User))] public interface IUserRepository { [Query("SELECT * FROM users WHERE id = @id")] Task<User?> GetById(int id); }

ICrudRepository<TEntity, TKey>

Provides 8 standard CRUD operations with zero configuration.

[RepositoryFor(typeof(Product))] public interface IProductRepository : ICrudRepository<Product, int> { // GetById, GetAll, Insert, Update, Delete, Count, Exists, GetPage }

SQL Template Placeholders

40+ placeholders for dynamic SQL generation.

{{table}} // Table name {{columns}} // All columns {{values}} // Parameter placeholders {{where}} // WHERE clause {{set}} // SET clause for UPDATE {{orderby}} // ORDER BY {{limit}} // LIMIT/TOP {{startswith}} // LIKE prefix% {{contains}} // LIKE %text% {{json_extract}}// Extract JSON {{upsert}} // INSERT ... ON CONFLICT {{date_add}} // Add to date // ... and 30+ more

üöÄ Quick Start

1. Install

dotnet add package Sqlx

2. Create Repository

[RepositoryFor(typeof(User))] [SqlDialect(SqlDefine.PostgreSql)] public interface IUserRepository : ICrudRepository<User, int> { [Query("SELECT * FROM users WHERE email = @email")] Task<User?> GetByEmail(string email); }

3. Use It!

services.AddScoped<IUserRepository, UserRepositoryImpl>(); var user = await repo.GetById(1); var users = await repo.GetAll(); await repo.Insert(new User { Name = "John" });
