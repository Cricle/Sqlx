📋 项目概览

Sqlx 是什么？它能为你做什么？

🎯 核心理念

  • 不写列名 - 自动从实体类生成
  • 零反射 - 编译时代码生成
  • 类型安全 - 编译时检查
  • 高性能 - 接近手写 ADO.NET
  • 易维护 - 添加字段自动更新
✅ 适用场景： CRUD密集型应用、微服务、Web API、需要多数据库支持的项目

📊 性能数据

  • 单行查询：7.371μs（vs Raw 6.434μs）
  • 内存分配：1.21KB（比Dapper少46%）
  • 零反射开销
  • 完整Activity追踪
  • 比Dapper快20%
ℹ️ 性能优势： 通过编译时代码生成和多项优化，Sqlx比Dapper快20%，同时提供完整的分布式追踪
