## 📊 项目亮点

- **⚡ 高性能**: 比 Dapper 快 37%，内存少 48%
- **🧪 高质量**: 3,738 个测试，88.6% 覆盖率
- **🔒 类型安全**: 编译时验证，零运行时错误
- **🌐 多数据库**: 支持 SQLite, PostgreSQL, MySQL, SQL Server
- **📦 零配置**: 开箱即用，无需复杂配置
