# 📁 项目结构

```
Sqlx/
├── src/Sqlx/              # 核心库
├── tests/                 # 测试项目
├── samples/               # 示例代码
```

## 核心组件

- **Source Generator** - 编译时代码生成
- **Repository Pattern** - 自动接口实现  
- **ExpressionToSql** - LINQ 表达式转 SQL
- **BatchCommand** - 批量操作支持
