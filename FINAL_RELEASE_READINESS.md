### NuGet 包发布
```bash
# 构建发布版本
dotnet pack src/Sqlx/Sqlx.csproj --configuration Release

# 发布到 NuGet (需要 API Key)
dotnet nuget push packages/Sqlx.2.0.1.nupkg --source https://api.nuget.org/v3/index.json
```
