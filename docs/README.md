## 📝 文档约定

本文档使用以下约定：

- **占位符** - 用 `{{}}` 包裹，如 `{{columns}}`
- **参数** - 用 `@` 前缀，如 `@id`
- **选项** - 用 `--` 前缀，如 `--exclude`
- **代码示例** - 使用代码块展示
- **最佳实践** - 用 ✅ 标记
- **避免做法** - 用 ❌ 标记
